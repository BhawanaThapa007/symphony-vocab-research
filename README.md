# Symphony Vocabulary Learning System

AI-Enhanced Gamified Vocabulary Learning for Grade 4 EFL Students

## Overview

This is a research-based vocabulary learning system implementing:
- **CTML** (Cognitive Theory of Multimedia Learning)
- **SDT** (Self-Determination Theory)
- **ZPD** (Zone of Proximal Development)
- **Vocabulary Learning Theory** (Nation, Beck)
- **DGBL** (Digital Game-Based Learning)

## Quick Start

### For Students:
1. Open the website
2. Click "Student"
3. Enter your name
4. Start learning!

### For Teachers/Researchers:
1. Open the website
2. Click "Admin"
3. Login: `admin` / `admin123`
4. View data and export CSV files

## Features

- ✅ 15 vocabulary words (Unit 1: School & Learning)
- ✅ AI adaptive learning (ZPD-based)
- ✅ Gamification (points, badges, levels, streaks)
- ✅ Automatic progress saving
- ✅ Research data collection
- ✅ CSV export for SPSS/Excel analysis
